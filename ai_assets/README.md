# AI Assets Repository 🧠

Welcome to the central storage for the **Smart Farm AI** intelligence assets.

## 📂 Structure

- **`animal_weights/`**: Dedicated storage for machine learning models and weight data for various livestock (Bees, Cows, Poultry, etc.).
- **`anomaly_detection/`**:
  - **`fire_hazards/`**: Specialized models for detecting environmental risks like fire or smoke using sensor data and computer vision.

## 🛠️ Usage

Place your `.h5`, `.pt`, `.json`, or `.weights` files in the corresponding subdirectories to ensure the system can correctly load and utilize them for real-time monitoring.
